== Description ==

Credit Calc it is a plugin for credit company. The goal of plugin is provide for the client convenient method of payment
on the company site.

Credit calc plugin includes the following functions:

* Select credit sum and amount
* Calculate the sum per month
* Sending order data to admin email
* Output the form from short code [credit_calc]
* Settings of form

Have a suggestion or found a bug? [Let me know](mailto:kostikovmu@ya.ru)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/credit_calc` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin
4. Add shortcode [credit_calc] to the required location on the page or post
